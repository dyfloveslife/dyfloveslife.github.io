打开服务器管理器，右击角色，然后添加角色，选中Web服务器(IIS)，然后一直下一步，直至安装完成。如下图：

三. 个人目录

1.打开Internet信息服务(IIS)管理器，选中网站下的DefaultWebSite，在右侧操作中，选择基本设置。物理路径设置为：c:\inetpub\wwwroot即可。如下图：

2.右击DefaultWebSite，选择网站绑定→添加。IP地址选择已经配置的静态IP：192.168.1.2，主机名随意填写：www.dyf.com。如下图：

3.在浏览器输入：http://localhost。如下图：

4.在 C 盘下，新建dyf.com文件夹，并在该文件夹下新建myindex.html文件，填入内容。

5.打开Internet信息服务(IIS)管理器，右击网站→添加网站，填入网站名称、物理路径、IP地址。如下图：

6.在dyf的中间一栏，有目录浏览，双击它。在右侧操作，单击启用。如下图：

7.在浏览器输入：http://192.168.1.2即可。

四. 虚拟目录

1.在 C 盘的dyf.com文件夹下，新建爱好文件夹，并在该文件夹下创建 myindex.html 文件，写入任意文字。

2.打开Internet 信息服务(IIS)管理器，右击dyf→添加虚拟目录。在别名一栏填入爱好，物理路径为：c:\dyf.com\爱好。如下图：

3.在浏览器输入：http://192.168.1.2即可看到爱好一栏。

五. 创建虚拟主机

1. 相同 IP 地址，不同TCP端口

我们之前已经用http://192.168.1.2这一IP地址添加了网站，其默认端口号是80。所以，当我们在浏览器访问http://192.168.1.2时，就相当于访问http://192.168.1.2:80,也就是说默认端口号80是可以省略的。

接下来，我们也是用http://192.168.1.2这一IP地址添加网站。只不过将其端口号设置成8080，在访问时，应访问http://192.168.1.2:8080。

1.先在 C 盘下，新建webB 文件夹，并在里面新建 myindex.html 文件，写入任意文字。如下图：

2.打开Internet 信息服务(IIS)管理器，右击网站→添加网站。网站名称可随意填写: web_B，物理路径：c:\webB，IP 地址：192.168.1.2，端口：8080。如下图：

3.在web_B的中间一栏，有目录浏览，双击它。在右侧操作，单击启用。

4.在浏览器输入:http://192.168.1.2:8080，即可访问。

2. 不同IP地址，相同 TCP 端口

1. 添加 IP 地址

单击 开始→控制面板→网络和共享中心→本地连接→属性→Internet 协议版本4（TCP/IPv4）→高级→IP设置→添加，IP地址为:192.168.1.3，子网掩码为：255.255.255.0。最后点击 添加 即可。如下图：

2. 绑定 IP 地址

1.打开 Internet 信息服务(IIS)管理器，右击 dyf→ 网站绑定。点击 添加 。IP地址选择为：192.168.1.3。如下图；

2.重新启动该网站。

3.在浏览器输入 http://192.168.1.3，即可访问。如下图：
