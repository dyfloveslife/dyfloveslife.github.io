Windows 环境下配置 DNS 服务器

2017-06-04
Windows
587 words 2 mins read

Contents

软件： VMware Workstation Pro

环境： Windows 2008 R2

步骤：

  1. 配置虚拟机静态 IP
  2. 安装 DNS
  3. 创建正向查找区域
  4. 创建反向查找区域
  5. 测试

一. 配置虚拟机静态 IP

1
2
3

IP 地址：192.168.1.2
子网掩码：255.255.255.0
首选 DNS 服务器：192.168.1.2

二. 安装

打开服务器管理器，右击角色，然后添加角色。选中DNS服务器。如下图：

然后一直下一步，直至安装完成。

三. 正向查找区域

1.点击开始→管理工具→DNS，右击正向查找区域→新建区域.选择主要区域，然后填入区域名称：dyf.com。下一步，选择不允许动态更新。如下图：

最后单击完成即可。

2.右击正向查找区域下的dyf.com，选择新建主机，名称为：www，IP 地址为你的虚拟机的静态 IP 地址。

四. 正向测试

开始→运行，键入 cmd.在dos界面下，输入nslookup.接着输入www.dyf.com，若出现对应IP地址，即可完成。如下图：

五. 反向查找区域

1.右击反向查找区域→新建区域，选择 主要区域，选择 IPv4 反向查找区域，网络填写：192.168.1，一直下一步，直至完成即可。

2.右击反向查找区域下的1.168.192.in-addr.arpa，选择新建指针。在主机名一栏，点击浏览。如下图：

3.一直双击下图红框区域，直至完成。

六. 反向测试

开始→运行，键入cmd，在 dos 界面下，输入nslookup。接着输入192.168.1.2，若出现对应域名，即可完成。如下图：
