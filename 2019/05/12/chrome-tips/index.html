Chrome 浏览器使用小贴士

2019-05-12
技巧
388 words 1 min read times read

Contents

  • 一、多线程下载
  • 二、加速视频缓冲速度

本篇文章主要介绍以下内容： 1. 开启 Chrome 浏览器多线程下载； 2. 加速 Youtube、Netflix 等流媒体视频播放。

一、多线程下载

当你使用 Chrome 浏览器下载资源的时候，难免会遇到下载缓慢的问题，这是因为 Chrome 浏览器默认没有开启多线程下载功能。若要开启该功能，只需要在地址栏里输入：

1

chrome://flags

然后在搜索框里输入：Parallel downloading ，点击 Enabled ，这时会提示你重新启动浏览器。该方法可以稍微的增加文件下载速度，不妨试试看。

二、加速视频缓冲速度

当你观看 Youtube 或 Netflix 等流媒体的时候，Chrome 会试图使用 QUIC 协议进行连接，或将部分 TCP 链接转换为 QUIC 。此时遇到运营商限制 UDP，观看视频的体验会非常差，缓冲和加载速度会被大幅度影响，所以在这里我们可以关闭 QUIC 协议。 具体操作也是和上面的一样，在 Chrome 的地址栏输入：

1

chrome://flags

然后在搜索框里输入：Experimental QUIC protocol，将其设置为 Disable ，这时会提示你重新启动浏览器，重启配置生效。
